## üß† About Bubbles

Bubbles: Mind Physics is an interactive, physics-based educational game designed to teach the intellectual standards for critical thinking through engaging gameplay. I built this project specifically for my nephew Ntiska, who has a natural curiosity about how to think better but needed something more engaging than traditional learning methods.

The game uses realistic physics simulations (gravity, momentum, collisions) to create an immersive environment where players can develop critical thinking skills while having fun. Each level introduces a different intellectual standard, challenging players to apply these concepts through innovative gameplay mechanics.


## ‚ú® Features

- **8 Progressive Levels** - Each teaching one of the fundamental intellectual standards:
  - **Clarity** - Navigate through simple obstacles
  - **Accuracy** - Hit targets with precision
  - **Precision** - Make exact movements

- **Visual Feedback Systems** - Particle effects, animations, and color coding
- **Progress Tracking** - Monitor improvement across multiple play sessions
- **Responsive Design** - Play on desktop, tablet, or mobile devices
- **Dark Mode Support** - Easy on the eyes for extended play sessions

## üéÆ How It Works

The game uses a simple but powerful physics system built with vanilla JavaScript. Players drag and release to launch balls towards targets, navigating through various obstacles that represent barriers to clear thinking. Each level introduces new mechanics that correspond to the intellectual standard being taught.

For example, in the "Relevance" level, players must choose the correct ball (relevant information) to hit the target, while in the "Breadth" level, they need to hit multiple mini-targets (consider multiple perspectives) before reaching the main goal.
